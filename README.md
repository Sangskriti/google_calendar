# google_calendar

This project is a simple Google Calendar integration built using React. It allows users to view, add, and manage events through a clean and responsive interface. The app connects with Google Calendar using API authentication, enabling automatic syncing of events in real time. Users can create events, set reminders, edit or delete them, and view schedules in different formats such as day, week, or month. The project focuses on ease of use, fast rendering, and organized event management, making it a lightweight and user-friendly calendar solution built entirely with React.

run the project
npm run dev
